# EcoSphere 🌍

Track your carbon footprint, earn rewards, and adopt eco-friendly habits in a gamified way. EcoSphere helps you monitor your lifestyle impact and make a change.

## 🌟 Features
- **Track:** Log daily activities (e.g., electricity, transport, shopping) and calculate your carbon footprint automatically.
- **Rewards:** Earn eco points, unlock badges, and compete globally.
- **Suggestions:** Get personalized tips to improve your sustainability impact.
- **Marketplace:** Discover and buy eco-friendly products via zero-waste stores.
- **Community:** Join groups and participate in green challenges.
- **Carbon Credits:** Understand and offset your remaining emissions. 
- **Family Leaderboard:** Invite your family and see who is the most eco-friendly.

## 🛠️ Tech Stack
- **HTML, CSS (TailwindCSS)**: Built with vanilla front-end technologies focusing on speed and responsiveness.
- **Chart.js**: Dynamic and interactive visual data breakdown of your carbon footprint.
- **Lucide Icons & FontAwesome**: Beautiful, lightweight UI icons.

## 🚀 How to Run Locally

You can run this visually directly from your file system. No local server is strictly required.

1. Clone the repository:
   ```bash
   git clone <repository_url>
   ```
2. Navigate to the project directory:
   ```bash
   cd hackathon-main/hackathon-main
   ```
3. Open `index.html` in your favorite browser.

## 📦 Directory Structure
- `/pages` - Contains all dashboard, feature, and interaction views (login, dashboard, rewards, etc.).
- `/assets` - Visual resources including product placeholder images.
- `index.html` - Minimal landing page.

## 💡 About
We believe small actions create big impacts. EcoSphere empowers individuals to measure, understand, and reduce their carbon footprint in an engaging, gamified way. Join us in making sustainability fun and impactful 🌍.
